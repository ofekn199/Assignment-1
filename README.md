# Assignment-1

A simple Python program for Assignment 1 that demonstrates basic programming concepts including:
- Functions
- String formatting
- Input/output operations
- Documentation

## Requirements

- Python 3.6 or higher

## Usage

Run the program using Python:

```bash
python3 assignment1.py
```

The program will:
1. Display a welcome message
2. Print a "Hello, World!" greeting
3. Ask for your name
4. Greet you personally

## Running Tests

To run the tests:

```bash
python3 test_assignment1.py
```
